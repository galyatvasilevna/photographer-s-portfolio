# Веб-интерфейс для портфолио фотографа

## 1. Аннотация

Проект представляет собой одностраничный веб-сайт, разработанный для демонстрации портфолио профессионального фотографа. Реализованы динамическая фильтрация галереи, переключение цветовых тем (светлая/тёмная/автоматическая), форма обратной связи с клиентской валидацией и сохранением черновика в `localStorage`, а также адаптивная вёрстка для корректного отображения на устройствах с различным разрешением экрана. Серверная часть отсутствует; все данные обрабатываются на стороне клиента.

## 4. Инструкция по локальному развертыванию

Для выполнения кода проекта на локальной машине необходимо выполнить следующую последовательность действий:

1. **Клонирование репозитория**  
   Выполнить команду в терминале:
   ```bash
   git clone https://github.com/username/photographer-portfolio.git
   ```
   (где `username` — имя владельца репозитория)

2. **Переход в директорию проекта**
   ```bash
   cd photographer-portfolio
   ```

3. **Открытие веб-сайта**  
   - Запустить файл `index.html` в любом современном веб-браузере (Google Chrome, Mozilla Firefox, Safari, Microsoft Edge).  
   - Для удобства разработки рекомендуется использовать локальный веб-сервер, например, расширение **Live Server** для VS Code.

## 5. Руководство пользователя

### 5.1 Навигация
- На десктопных устройствах меню расположено в левой части экрана, содержит пункты: «Главная», «Портфолио», «Об авторе», «Услуги», «Контакты».
- При ширине экрана менее 768 пикселей меню трансформируется в «гамбургер»-меню (иконка с тремя горизонтальными линиями в правом верхнем углу).
- Клик по пункту меню осуществляет плавную прокрутку к выбранной секции.

### 5.2 Просмотр портфолио и фильтрация
- Секция «Портфолио» содержит галерею изображений.
- Над галереей расположены кнопки фильтрации по категориям: «Все работы», «Свадьба», «Портреты», «Fashion», «События», «Коммерция», «Пейзажи», «Street».
- При выборе категории галерея динамически перестраивается, отображая только изображения, соответствующие выбранному тегу.
- В случае отсутствия работ в категории выводится сообщение «Нет работ в этой категории».

### 5.3 Полноэкранный просмотр изображений
- Клик по любому изображению в галерее открывает модальное окно с увеличенной версией фотографии.
- Навигация между изображениями осуществляется с помощью кнопок «‹» (предыдущее) и «›» (следующее), а также клавишами‑стрелками на клавиатуре.
- Закрытие модального окна возможно по кнопке «×» (крестик), клику на тёмный фон или нажатию клавиши Escape.

### 5.4 Управление темой оформления
- В правой части меню расположен блок переключателей темы: «🌞 Светлая», «🌙 Тёмная», «⚡ Авто».
- При выборе темы «Авто» система автоматически определяет системные настройки операционной системы (через медиавыражение `prefers-color-scheme`) и применяет соответствующую цветовую схему.
- Выбранная тема сохраняется в `localStorage` и восстанавливается при последующих посещениях сайта.

### 5.5 Форма обратной связи
- Секция «Контакты» содержит форму с полями: «Имя» (обязательное, минимум 2 символа), «Email» (проверка наличия символов «@» и «.»), «Телефон» (необязательное, валидация количества цифр), «Сообщение» (обязательное, минимум 10 символов).
- При частичном заполнении поля данные автоматически сохраняются в `localStorage` как черновик.
- При нажатии кнопки «Отправить сообщение» выполняется клиентская валидация:
  - При обнаружении ошибок под соответствующими полями отображаются текстовые сообщения, а также всплывающее уведомление.
  - При успешной проверке данные выводятся в консоль браузера (имитация отправки), форма очищается, черновик удаляется, пользователь получает уведомление об успехе.

### 5.6 Информационные секции
- **Об авторе** – содержит текстовую биографию, профессиональное изображение фотографа, список достижений и публикаций.
- **Услуги и цены** – представлены три основных пакета (свадебная, портретная, коммерческая съёмка) и три дополнительные услуги. Каждая карточка содержит кнопку «Заказать», которая выполняет плавную прокрутку к форме обратной связи.
